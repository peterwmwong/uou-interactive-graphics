
# Controls

| Mouse                          | Action                                       |
|--------------------------------|----------------------------------------------|
| Right button drag              | Camera zoom in/out                           |
| Left button drag               | Camera orbits                                |

| Key | Action                                                 |
|-----|--------------------------------------------------------|
|  P  | Toggle between Perspective and Orthographic projection |
