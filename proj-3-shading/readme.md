# Usage

```sh
cargo run --bin proj-3-shading
```

# Controls

| Mouse                          | Action                                       |
|--------------------------------|----------------------------------------------|
| Right button drag              | Camera zoom in/out                           |
| Left button drag               | Camera orbits                                |
| Ctrl + Right button drag       | Light orbits                                 |

| Key | Action                                 |
|-----|----------------------------------------|
|  0  | Ambient + Diffuse + Specular (default) |
|  1  | Normals                                |
|  2  | Ambient                                |
|  3  | Ambient + Diffuse                      |
|  4  | Specular                               |
